# 📂 演示报告目录 (Demo Reports Directory)

本目录用于存放电商增长 Agent 自动生成的各类分析报告示例（支持 PDF 打印导出、Markdown 导出以及 JSON 结构化数据示例）。

## 📄 演示报告分类建议
你可以将手动导出的以下报告示例保存于本目录中进行开源展示：

1. **📡 趋势雷达报告 (Event-Driven Trend Radar)**
   - 示例：针对“欧洲高温空调脱销”等突发气候事件挖掘的周边长尾需求与避坑选品报告。
2. **🌍 跨境选品探索 (Etsy Cross-Border Blueprints)**
   - 示例：挖掘 Etsy 平台高溢价手工/定制礼品的蓝海选品方案。
3. **🍊 半托管风控评估 (Temu Semi-Managed Audit)**
   - 示例：针对低毛利备货、末端仓储与配送费用线进行的安全毛利审计报告。
4. **🎵 社交爆量挖掘 (TikTok Shop Trend Analysis)**
   - 示例：通过 TikTok 热门短视频脚本元素推演出的高转化贴牌商品蓝图。

## 📥 如何导出示例？
1. 打开侧边栏插件，配置好 LLM 密钥，选择并运行任意专家 Skill。
2. 执行完成后，在结果区域：
   - 点击 **【📥 下载报告 (PDF)】**：在弹出的预览窗口中选择“另存为 PDF”。
   - 点击 **【📥 导出 Markdown】**：导出标准格式的 `.md` 文档。
3. 将导出的文件重命名并保存至本目录中。
